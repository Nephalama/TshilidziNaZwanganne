# Tshilidzi Na Zwangane (PTY) LTD Website
https://nephalama.github.io/TshilidziNaZwanganne/

This is the official company website for **Tshilidzi Na Zwangane (PTY) LTD**, a South African enterprise specializing in tender support, procurement, and supply chain solutions.

## 🚀 Features

- **Clean, Professional Design:** Built with a modern aesthetic using the Poppins font family.
- **Service Showcase:** Highlights key business areas including Procurement, Hygiene Products, PPE, and Tender Support.
- **Integrated Contact Forms:** Uses Formspree to handle inquiries without a backend server.
- **WhatsApp Integration:** A floating button allows instant customer communication.
- **Fully Responsive:** Designed to work seamlessly on desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

- HTML5
- CSS3 (linked via `/css/style.css`)
- JavaScript (linked via `/js/main.js`)
- [Formspree](https://formspree.io/) for form handling
- [Google Fonts](https://fonts.google.com/) (Poppins)
